# 巨正源项目目录结构及介绍
````
.nuxt  ---------  nuxt 框架目录
assets  --------  资源目录
components  ----  组件目录
layouts  -------  布局目录
middleware  ----  中间件目录
pages  ---------  页面目录
plugins  -------  插件目录
static  --------  静态文件目录
store  ---------  Store 目录
.nuxt.config.js   nuxt配置
package.json  --  包文件
````


##安装部署
1，安装  
npm install  
&   
cnpm i

2，进入环境

npm run dev

3, 部署
npm run build  
start
